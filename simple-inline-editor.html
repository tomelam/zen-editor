Simple Inline Editor and Run-Edit-Modal-Web-Page Demo

This Zen-enabled page is in Run Mode (its normal mode with all event handlers ready) until you press the ESCAPE key. You can verify this by clicking the inline-registered click-event handler for this button:

Did you click the button? If so, you should have seen an alert saying "We are in Run Mode". Now please cancel the alert if you have not already. Next, when you press the ESCAPE key, the web page flips into Edit Mode, where the page's controls and events are locked out and you can only interact with the page through Zen's control mechanisms. Try it: press the ESCAPE key, then click on an image. You will be presented with a Dialog with which you can edit an image's URL, figure caption, and subtext.
